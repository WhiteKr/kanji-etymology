## 1. 전체 공통 — JLPT 레벨 기준

JLPT는 공식 한자 목록을 공개하지 않아 통용 목록(jisho / Tanos / Waller 계열) 간
차이가 있다. 초안 데이터는 대체로 jisho 계열 기준이었다:

- 口·目·手·足을 **N4**로 분류 (Tanos 계열은 N5로 두기도 함)
- 好·門 → N3 (출처에 따라 門을 N2로 보기도)
- 竹·虫·糸 → N2, 貝 → N1 (목록마다 가장 크게 갈림)

**결정**: Tanos 목록으로 통일 완료 (2026-07-12). 전체 데이터 재정렬 반영됨.

## 2-1. 부수 서술 확장분 검수 포인트 (2026-07-12, Phase 2 — 2차 검수 대상)

부수 8개 본문을 어원 서술로 확장하면서 작성 에이전트가 자신도 낮음으로 표시한 부분:

| 부수 | 포인트 |
|------|--------|
| 冖 | 冪이 冖의 뜻을 잇는다는 서술, 수학 용어 "멱" 연결 — 통설 수준, 완충 표현 유지. 軍 윗부분 "전차를 둘러 가린 진영" 풀이는 여러 해석 중 하나 |
| 月 | 服의 月이 舟 유래라는 서술 (학설 대립, "견해도 있어"로 완충). 閒의 月 "달빛" 풀이도 통설이나 확정 아님 |
| 穴 | "갑골문 단계의 확실한 자형이 드물다" — 갑골문 존부 자체 재확인 권장. 突 "굴에서 개" 풀이는 說文 기반 통설 |
| 儿 | "어진사람" 명칭 유래 불분명 — 의도적으로 "여러 설명이 있다" 처리 |
| 日 | 원 안의 점 = "빈 동그라미와의 구별 표시" 견해 채택 — 대안 해석은 민간어원 우려로 배제 |
| 木 | 강희자전 소속 글자 수 "최상위권" 표현 — 정확한 순위 미검증 |

## 3. 說文解字 권수 표기

부수 배속 기준으로 기재했으며 판본에 따라 권 구분이 다를 수 있음 (특히 小 卷二,
南 卷六). 정밀 검수 시 통용 판본(段注 등) 기준으로 재확인 권장.

**웹 교차 검증 완료 (2026-07-12)** — 위키문헌 大徐本 원문(권별 페이지에서 표제자
직접 확인) + shuowen.org·詞典網 등 2차 소스:

| 한자 | 부수 배속 | 확인 결과 |
|------|-----------|-----------|
| 小 | 小部 | 卷二 — 현행 표기 일치 |
| 南 | 𣎵部 | 卷六 — 현행 표기 일치 |
| 東 | 東部 | 卷六 — 현행 표기 일치 |
| 好 | 女部 | 卷十二 — 현행 표기 일치 |
| 間 | 門部(閒 "隙也") | 卷十二 — 현행 표기 일치 |
| 学 | 教部(斅 "覺悟也", 學은 篆文 省體) | 卷三 — 현행 표기 일치 |
| 花 | 說文 미수록 (원자 華는 華部 卷六 "榮也") | ref 표기 조정 완료 |

주의: ctext.org는 서(敘)를 卷一로 세는 +1 오프셋 체제라 겉보기 권차가 하나씩
밀려 보임 (예: ctext "卷三 小部" = 大徐本 卷二). 교차 검증 시 혼동 주의.
