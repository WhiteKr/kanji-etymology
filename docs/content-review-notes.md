AI 초안 생성 시 작성 에이전트들이 스스로 자신도가 낮다고 표시한 부분의 목록.
검수 시 이 항목부터 확인하면 효율적이다.

## 1. 전체 공통 — JLPT 레벨 기준

JLPT는 공식 한자 목록을 공개하지 않아 통용 목록(jisho / Tanos / Waller 계열) 간
차이가 있다. 현재 데이터는 대체로 jisho 계열 기준:

- 口·目·手·足을 **N4**로 분류 (Tanos 계열은 N5로 두기도 함)
- 好·門 → N3 (출처에 따라 門을 N2로 보기도)
- 竹·虫·糸 → N2, 貝 → N1 (목록마다 가장 크게 갈림)

**결정 필요**: 프로젝트 표준 목록을 하나 정해 `docs/`에 명시하고 전체 재정렬.

## 2. 글자별 검수 포인트

| 한자 | 포인트 |
|------|--------|
| 学 | 갑골문 단계에 子 부재 → 금문에서 추가 서술 — 통설이지만 세부 연대 재확인 |
| 明 | 朙(囧+月) 기원설 vs 日+月 병존 — 갑골문 자형 병존 여부 자료마다 다름 |
| 中 | 갑골문 묘사가 깃발설 기준으로 서술됨 (학설 대립은 명시했으나 묘사 자체의 균형) |
| 上·下 | "짧은 선 → 세로획+점" 자형 정리 서술이 篆→隸 단계를 뭉뚱그림 (丄/丅 생략) |
| 手 | "갑골문에 手 없음, 又가 손 역할" — 통설이나 단정적 표현 재확인 |
| 足 | components의 口를 /kanji/口로 링크함 — 의미상 무관("자형 수렴"은 본문에 명시). 링크 유지 여부 결정 |
| 金 | 부품 표기에 丷 대용 사용 — 표기 관례 통일 여부 |
| 耳 | words의 初耳·耳元이 공식 JLPT 어휘 밖일 가능성 |
| 車 | related의 田("조감 구도 짝")은 어원 관계 아님 — 교육적 연결. 제거 검토 |
| 間 | evolution의 閒→間 시대 배치(전서/정자)가 도식적 — 실제로는 장기 공존 |
| 石 | 음독 コク(石高 단위) 포함 여부 — 학습 사전 성격상 제거 검토 |
| 貝 | 음독 バイ 단독 — KANJIDIC2 기준이나 실사용 빈도 극히 낮음 |
| 白 | 面白い 어원("눈앞이 환해진다")은 민간어원 성격 — 완충 표현으로 처리됨 |
| 花 | 說文 미수록 후기자라 sources ref를 "卷六(華)"로 표기 — 표기 방식 확인 |

## 3. 說文解字 권수 표기

부수 배속 기준으로 기재했으며 판본에 따라 권 구분이 다를 수 있음 (특히 小 卷二,
南 卷六). 정밀 검수 시 통용 판본(段注 등) 기준으로 재확인 권장.
